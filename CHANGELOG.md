All notable changes to this project will be documented in this file.

## [1.1.0] - 2021-08-24
* Infer repo URL from `package.json` or `.git/config` if it can't be

* Support autolinking repos references like `foo/bar#42`.
* Add tests.

## [1.0.1] - 2021-06-15
